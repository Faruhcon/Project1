Project1.html pulls the js files together.
	All javascript files are found in Project1/js/
		This includes: ball.js, constants.js, ghost.jsm
					   Main.js, maze.js, pacman.js
	
Main.js = "main method", contains the game loops
constants.js = constants used in game (grid layout, positions, text, etc)
ball.js = contains definition for the coins littered around the game
ghost.js = definition for the ghost objects
maze.js = definition for the game's maze
pacman.js = definition for pacman

All txt files are found in Project1/txt/
	This includes: GRADING.txt,README.txt,REPORT.txt
Screnshots are found in Project1/image/
	This includes: SCREENSHOT.PNG
